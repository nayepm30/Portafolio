Magic Cookies

Magic Cookies es un sistema web integral de e-commerce para la venta de galletas en línea. El proyecto cuenta con dos vistas principales: la del cliente, que permite explorar el catálogo de productos, agregar galletas al carrito, revisar precios en tiempo real y realizar pedidos de forma sencilla, y la del administrador, que ofrece herramientas para gestionar productos, controlar inventarios, actualizar precios y dar seguimiento a los pedidos. De esta manera, Magic Cookies cubre tanto la experiencia de compra de los usuarios como la gestión interna del negocio, ofreciendo una solución completa y funcional para un pequeño comercio en línea.

Tecnologías utilizadas

  • Python (Flask) como framework backend
  • MySQL como base de datos
  • HTML5, CSS3 y JavaScript
  • Bootstrap y Tailwind CSS para estilos
